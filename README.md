# parser_for_habr
<h3> Парсер для сайта Habr.com.<h3>
  
  
Собирает и сохраняет в БД список статей, далее в статьях собирает информацию: 
- Заголовок
- Дата
- Ссылка на пост
- Имя автор
- Ссылка на автора
- Ссылка на пост.
-------------------
Есть проблемы с ошибками кодировки при сборе материала статей, скоростью работы парсера.
